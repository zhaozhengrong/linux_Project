# 环境搭建
## 1.安装opencv库
```base
  sudo apt install libopencv-dev
```

## 2.下载NCNN源码
```base
  git clone https://github.com/Tencent/ncnn.git
```
创建编译目录
```base
  mkdir build
  cd build
```
进入build 执行`cmake .` 这里只是生成编译规则，还没有真正编译
```base
  cmake ..
```
然后开始编译 将`/src` 路径下的源码编译为静态库 `.a`
```base
  make -j$(nproc)
```

此时编译完之后项目需要包含
1. 源码头文件（src/*.h）这些是 NCNN 作者写的头文件。
2. 编译时生成的头文件（build/src/*.h）这里面的文件不是手写的，而是 CMake 在编译过程中自动生成的。
3. 静态库（libncnn.a）
